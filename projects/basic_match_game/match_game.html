Basic Memory Card Game

Start Date: November 5th, 2025 - Finish Date: November 5th, 2025



Game Info

This solo project is meant to be a fun little challenege to kill some time so to make it harder I decided to set a 24-hour limit to build a playable memory match game, aiming for something fun, functional, and clean. The goal: flip cards, find pairs, and clear the board with as few attempts as possible.



Rules / Core Mechanics

  • Mechanic 1: Generates a grid of shuffled cards
  • Mechanic 2: lets the user flip cards when pressed
  • Mechanic 3: when user flips to matching cards they are removed


Development Plan / Concept

The game will generate a grid of flipped cards, the player can click two cards at once to flip them, if they match they are removed from the grid, if they dont match they are reflipped and the player gets to try again. the goal is to remove all cards from the grid. Amount of attempts are tracked and the goal is to finish the grid in the shortest amount of attempts as possible.

Card sprites made in Aseprite



Comments After Project

The hardest part about making this game was to track each cards value and remove them from the grid without making everything shift.

If I was to expand on this game I could add features like highscore tracking and more card types, but since this is meant to be just a fun little practice project I might not do that.
